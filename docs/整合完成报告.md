## 🔧 使用方法

### 后端 API

```javascript
// 获取雅思词汇真经的单词
GET /api/words?source=雅思词汇真经

// 获取特定分类
GET /api/words?category=01_自然地理

// 组合查询
GET /api/words?source=雅思词汇真经&category=01_自然地理
```

### 小程序配置

```javascript
// 选择词库
const vocabSource = 'zhenjing'; // 'cambridge' 或 'zhenjing'

// 选择分类
const category = '01_自然地理';

// 播放音频
const audioUrl = `${apiUrl}/audio/vocabulary/audio/${category}/${word}.mp3`;
```
