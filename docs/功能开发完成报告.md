### 2️⃣ API 扩展 - 支持按词库和分类查询

**新增路由** (`backend/routes/words.js`):
```javascript
GET /api/words/libraries          // 获取词库列表
GET /api/words/categories         // 获取分类列表
GET /api/words/library/:id/words  // 按词库和分类查询
GET /api/words/words/:id          // 获取单词详情
```

**控制器函数** (`backend/controllers/words.js`):
- `getLibraries()` - 查询所有词库及统计信息
- `getCategories()` - 按词库查询分类列表
- `getWordsByLibrary()` - 按词库和分类筛选单词
- `getWordDetail()` - 获取单词详情（含音频 URL）

**API 示例**:

```bash
# 获取词库列表
curl -H "Authorization: Bearer <token>" \
  https://caiyuyang.cn:3001/api/words/libraries

# 获取雅思词汇真经的分类
curl -H "Authorization: Bearer <token>" \
  "https://caiyuyang.cn:3001/api/words/categories?source=真经"

# 获取自然地理分类的单词
curl -H "Authorization: Bearer <token>" \
  "https://caiyuyang.cn:3001/api/words/library/zhenjing/words?category=01_自然地理"

# 获取单词详情
curl -H "Authorization: Bearer <token>" \
  https://caiyuyang.cn:3001/api/words/words/123
```

```
│   └── ...
└── ...
```

**小程序播放** (`frontend/pages/learning/learning.js`):
```javascript
// 播放单词发音（优先使用词汇音频）
playWordAudio() {
  const { currentWord, audioContext } = this.data;
  
  if (currentWord.category) {
    // 🆕 使用词汇音频
    audioContext.src = `${apiUrl}/audio/vocabulary/audio/${currentWord.category}/${word}.mp3`;
  } else {
    // 使用原有发音服务
    audioContext.src = `${apiUrl}/pronunciation/word-audio/${word}`;
  }
  
  audioContext.play();
}
```

## 📝 使用说明

### 用户操作流程

1. **选择词库**
   - 进入"配置"页面
   - 点击选择词库（剑桥雅思 1-18 或 雅思词汇真经）
   - 如选择真经词库，可进一步选择分类

2. **开始学习**
   - 系统根据选择的词库和分类推送单词
   - 学习页面显示词库和分类标签
   - 点击🔊播放单词发音

3. **音频播放**
   - 自动播放：进入单词时自动播放
   - 手动播放：点击🔊按钮播放
   - 章节音频：可播放整章音频（规划中）
