## 🔍 错误原因

### WXML 不支持 JavaScript 可选链操作符

微信开发者工具的 WXML 模板引擎**不支持** ES2020 的可选链操作符 (`?.`)。

**错误写法**:
```xml
{{categories[categoryIndex]?.name || '全部章节'}}
```

**正确写法**:
```xml
<block wx:if="{{categoryIndex >= 0 && categories.length > categoryIndex}}">
  {{categories[categoryIndex].name}}
</block>
<block wx:else>
  全部章节
</block>
```

## ✅ 修复方案

### 修复 1: config.wxml - 可选链操作符

**修复文件**: `frontend/pages/config/config.wxml`

**修复前**:
```xml
<picker 
  mode="selector" 
  range="{{categories}}" 
  range-key="name"
  value="{{categoryIndex}}"
  bindchange="selectCategory">
  <view class="category-picker">
    {{categories[categoryIndex]?.name || '全部章节'}}
  </view>
</picker>
<view class="text" wx:if="{{categories[categoryIndex]}}">
  共 {{categories[categoryIndex].word_count}} 个单词
</view>
```

**修复后**:
```xml
<picker 
  mode="selector" 
  range="{{categories}}" 
  range-key="name"
  value="{{categoryIndex}}"
  bindchange="selectCategory">
  <view class="category-picker">
    <block wx:if="{{categoryIndex >= 0 && categories.length > categoryIndex}}">
      {{categories[categoryIndex].name}}
    </block>
    <block wx:else>
      全部章节
    </block>
  </view>
</picker>
<view class="text" wx:if="{{categoryIndex >= 0 && categories.length > categoryIndex && categories[categoryIndex].word_count}}">
  共 {{categories[categoryIndex].word_count}} 个单词
</view>
```

### 修复 2: learning.wxml - 字符串方法

**问题**: WXML 不支持 `replace()`、`substring()` 等 JavaScript 字符串方法

**解决方案**: 在 JS 中预处理数据

**修复文件**: `frontend/pages/learning/learning.js`

**修复前**:
```javascript
showNextWord() {
  const currentWord = words[currentIndex];
  this.setData({
    currentWord: currentWord,
    progress: ((currentIndex + 1) / totalWords) * 100
  });
}
```

**修复后**:
```javascript
showNextWord() {
  const currentWord = words[currentIndex];
  
  // 🆕 预处理分类名称（去掉编号前缀）
  let categoryName = '';
  if (currentWord.category) {
    categoryName = currentWord.category.replace(/^\d+_/, '');
  }
  
  this.setData({
    currentWord: {
      ...currentWord,
      categoryName: categoryName
    },
    progress: ((currentIndex + 1) / totalWords) * 100
  });
}
```

**WXML 使用**:
```xml
<!-- 修复前（错误） -->
<text>{{currentWord.category.replace(/^\d+_/, '')}}</text>

<!-- 修复后（正确） -->
<text>{{currentWord.categoryName}}</text>
```

## 📋 微信 WXML 语法注意事项

### ❌ 不支持的 JavaScript 语法

1. **可选链操作符** (`?.`)
   ```xml
   <!-- 错误 -->
   {{user?.name}}
   ```

2. **空值合并操作符** (`??`)
   ```xml
   <!-- 错误 -->
   {{value ?? 'default'}}
   ```

3. **模板字符串**
   ```xml
   <!-- 错误 -->
   {{`Hello ${name}`}}
   ```

4. **箭头函数**
   ```xml
   <!-- 错误 -->
   {{items.map(item => item.name)}}
   ```

5. **正则表达式方法** (`replace()`, `match()`, `test()`)
   ```xml
   <!-- 错误 -->
   {{text.replace(/^\d+_/, '')}}
   ```

6. **字符串方法** (`substring()`, `substr()`, `slice()`, `toUpperCase()`)
   ```xml
   <!-- 错误 -->
   {{text.substring(3)}}
   ```

7. **数组方法** (`map()`, `filter()`, `reduce()`)
   ```xml
   <!-- 错误 -->
   {{items.map(item => item.name)}}
   ```

### ✅ 推荐的 WXML 写法

1. **使用 `wx:if` 进行条件判断**
   ```xml
   <block wx:if="{{user && user.name}}">
     {{user.name}}
   </block>
   <block wx:else>
     匿名用户
   </block>
   ```

2. **使用 `wx:for` 进行列表渲染**
   ```xml
   <view wx:for="{{items}}" wx:key="id">
     {{item.name}}
   </view>
   ```

3. **使用三元运算符的替代方案**
   ```xml
   <!-- 使用 wx:if/else 替代 -->
   <text wx:if="{{score >= 60}}">及格</text>
   <text wx:else>不及格</text>
   ```

4. **在 JS 中预处理复杂数据**
   ```javascript
   // JS 中处理
   const categoryName = category.replace(/^\d+_/, '');
   this.setData({ categoryName });
   
   // WXML 中直接使用
   <text>{{categoryName}}</text>
   ```
