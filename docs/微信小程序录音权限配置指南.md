# 📋 微信小程序录音权限配置指南

**问题**: 用户反馈录音权限弹窗显示"使用信息记录"而不是"允许/拒绝"  
**原因**: 微信 2023 年隐私保护新规要求  
**更新时间**: 2026-03-30

---

## 🔍 问题原因

微信从 2023 年 9 月开始实施**隐私保护新规**：

| 旧版（2023 年前） | 新版（2023 年后） |
|------------------|------------------|
| 直接弹窗"允许/拒绝" | 先显示"使用信息记录" |
| 无需隐私协议 | 必须配置隐私协议 |
| `permission` 配置即可 | 需要 `permission` + `privacy` + 后台配置 |

---

## ✅ 完整解决方案

### 步骤 1: app.json 配置（已完成 ✅）

```json
{
  "permission": {
    "record": {
      "desc": "用于跟读练习功能，录制您的发音进行智能评分"
    }
  },
  "privacy": {
    "record": {
      "desc": "用于跟读练习功能，录制您的发音进行智能评分"
    }
  }
}
```

### 步骤 2: 微信开发者工具配置（⚠️ 必须手动操作）

1. **打开微信开发者工具**
2. 点击顶部菜单：**工具 → 隐私协议设置**
3. 在弹出的隐私配置页面中：
   - ✅ 勾选"录音"权限
   - 填写使用场景说明：`用于跟读练习功能，录制用户的发音进行智能评分，帮助用户提高口语水平`
   - 填写收集内容：`用户录音音频`
   - 填写用途：`发音评分和反馈`
4. 点击**保存**

### 步骤 3: 小程序管理后台配置（⚠️ 必须手动操作）

1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入你的小程序
3. 左侧菜单：**设置 → 基本设置 → 服务内容声明**
4. 添加"录音"功能：
   - 功能类型：录音
   - 使用场景：发音练习、口语评分
   - 说明文案：用于录制用户跟读发音，进行 AI 智能评分
5. 提交审核（如需要）

### 步骤 4: 代码中处理隐私授权（推荐）

在需要使用录音功能的页面（如 `learning.js` 和 `review.js`）添加：

```javascript
// 检查隐私授权状态
checkPrivacyAuthorization() {
  return new Promise((resolve, reject) => {
    // 微信基础库 2.32.0+ 支持
    if (wx.getPrivacySetting) {
      wx.getPrivacySetting({
        success: (res) => {
          if (res.needAuthorization) {
            // 需要用户授权
            wx.requirePrivacyAuthorize({
              success: () => {
                console.log('✅ 隐私授权成功');
                resolve(true);
              },
              fail: () => {
                console.log('❌ 隐私授权失败');
                wx.showModal({
                  title: '权限未授权',
                  content: '录音功能需要您授权隐私协议，请在小程序设置中开启',
                  confirmText: '去设置',
                  success: (res) => {
                    if (res.confirm) {
                      wx.openSetting({
                        withSubscriptions: true
                      });
                    }
                  }
                });
                resolve(false);
              }
            });
          } else {
            // 已授权或不需要授权
            resolve(true);
          }
        },
        fail: () => {
          // 基础库版本过低，直接尝试录音
          console.log('⚠️ 基础库版本过低，直接尝试录音');
          resolve(true);
        }
      });
    } else {
      // 基础库不支持，直接尝试录音
      resolve(true);
    }
  });
}

// 使用示例
async startRecording() {
  // 先检查隐私授权
  const authorized = await this.checkPrivacyAuthorization();
  if (!authorized) {
    return;
  }
  
  // 开始录音
  wx.startRecord({
    success: (res) => {
      // ...
    }
  });
}
```

## ⚠️ 常见问题

### Q1: 配置后还是显示"使用信息记录"
**A**: 这是正常的！新版微信就是这样的文案，关键是后面要有"允许/拒绝"选项。

### Q2: 用户拒绝后如何重新授权？
**A**: 引导用户去小程序设置：
```javascript
wx.openSetting({
  withSubscriptions: true
});
```

### Q3: 开发环境正常，线上环境不行
**A**: 需要在微信公众平台后台配置隐私协议，并提交审核（如需要）。

### Q4: 基础库版本要求
**A**: 
- `wx.getPrivacySetting`: 基础库 2.32.0+
- `wx.requirePrivacyAuthorize`: 基础库 2.32.0+
- 低版本需要降级处理

## 📊 兼容性处理

```javascript
// 兼容新旧版本
function checkRecordPermission() {
  // 新版本（2.32.0+）
  if (wx.getPrivacySetting) {
    return checkPrivacyNew();
  }
  // 旧版本
  else {
    return checkPrivacyOld();
  }
}

function checkPrivacyOld() {
  return new Promise((resolve) => {
    wx.getSetting({
      success: (res) => {
        // 检查录音权限
        if (res.authSetting['scope.record'] === undefined) {
          // 未询问过，直接返回 true（会让 wx.startRecord 触发授权）
          resolve(true);
        } else if (res.authSetting['scope.record'] === true) {
          // 已授权
          resolve(true);
        } else {
          // 已拒绝
          resolve(false);
        }
      }
    });
  });
}
```
