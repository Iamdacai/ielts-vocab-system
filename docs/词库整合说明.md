## 🔧 使用方法

### 后端 API

#### 1. 按词库来源查询

```javascript
// 获取雅思词汇真经的所有单词
GET /api/words?source=雅思词汇真经

// 获取特定分类的单词
GET /api/words?category=01_自然地理

// 获取特定分类的单词（按词库来源）
GET /api/words?source=雅思词汇真经&category=01_自然地理
```

#### 2. 数据库查询示例

```javascript
// 查询某个分类的所有单词
const words = await db.all(`
  SELECT * FROM ielts_words 
  WHERE source = '雅思词汇真经（刘洪波）'
  AND category = ?
  ORDER BY id
`, [category]);

// 统计各分类词汇量
const stats = await db.all(`
  SELECT category, COUNT(*) as count
  FROM ielts_words
  WHERE source = '雅思词汇真经（刘洪波）'
  GROUP BY category
  ORDER BY count DESC
`);
```

### 小程序前端

#### 1. 词库选择

在学习配置页面添加词库选择：

```javascript
// pages/config/config.js
const vocabSources = [
  {
    id: 'cambridge',
    name: '剑桥雅思 1-18',
    count: 4464,
    description: '基于剑桥雅思真题 1-18'
  },
  {
    id: 'zhenjing',
    name: '雅思词汇真经',
    count: 3674,
    description: '刘洪波著，按场景分类'
  }
];

// 用户选择词库后保存
wx.setStorageSync('selected_vocab_source', 'zhenjing');
```

#### 2. 分类学习

```javascript
// pages/learning/category-list.js
Page({
  data: {
    categories: []
  },
  
  onLoad() {
    const source = wx.getStorageSync('selected_vocab_source') || 'cambridge';
    if (source === 'zhenjing') {
      this.loadZhenjingCategories();
    }
  },
  
  async loadZhenjingCategories() {
    const res = await wx.request({
      url: `${getApp().globalData.apiUrl}/categories?source=雅思词汇真经`,
      method: 'GET'
    });
    
    this.setData({
      categories: res.data.map(cat => ({
        id: cat.category,
        name: cat.category.split('_')[1], // 去掉编号
        count: cat.count,
        audio: `/vocabulary/audio/${cat.category}.mp3`
      }))
    });
  }
});
```

#### 3. 音频播放

```javascript
// 播放章节音频
playChapterAudio(category) {
  const audio = wx.createInnerAudioContext();
  audio.src = `${getApp().globalData.apiUrl}/audio/vocabulary/audio/${category}.mp3`;
  audio.play();
}

// 播放单词音频
playWordAudio(category, word) {
  const audio = wx.createInnerAudioContext();
  audio.src = `${getApp().globalData.apiUrl}/audio/vocabulary/audio/${category}/${word}.mp3`;
  audio.play();
}
```

## 🎯 推荐学习路径

### 方案 A：场景分类优先（推荐基础薄弱）

1. **第一阶段**：选择"雅思词汇真经"词库
2. **按场景学习**：从熟悉的话题开始（如饮食健康、娱乐运动）
3. **逐步进阶**：学习抽象话题（如法律法规、社会经济）
4. **第二阶段**：切换到"剑桥雅思 1-18"，针对性刷题

### 方案 B：真题优先（推荐有基础）

1. **第一阶段**：选择"剑桥雅思 1-18"词库
2. **按册学习**：从剑 1 开始，逐步推进
3. **补充强化**：用"雅思词汇真经"按场景查漏补缺
