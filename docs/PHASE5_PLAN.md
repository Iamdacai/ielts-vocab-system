# Phase 5 开发计划 - 权威词库 + 拼写模式

## 📚 词库资源（来自 hefengxian/my-ielts）

### 已获取数据
1. **雅思词汇真经** - 刘洪波逻辑词群记忆法
   - 文件：`ielts-reference/src/pages/vocabulary/vocabulary.js`
   - 词数：约 3000+ 词
   - 分类：按逻辑词群分组

2. **听力 179 考点词**
   - 文件：`ielts-reference/src/pages/listening/listening179.json`
   - 词数：179 个高频考点词
   - 特色：含同义替换

3. **阅读 538 同义替换**
   - 包含在词汇真经数据中

### 数据格式
```javascript
{
  "word": ["inflation"],
  "pos": "n.",
  "meaning": "通货膨胀",
  "example": "...",
  "extra": "...",
  "spellError": false,  // 拼写错误标记
  "spellValue": "",     // 拼写答案
  "showSource": false   // 显示原词
}
```
