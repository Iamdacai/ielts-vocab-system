## 触发时机建议

**最佳时机**：用户登录后首次进入配置页

**实现方式**：
1. 检测用户配置中的 `vocab_library` 是否在旧词库列表中
2. 如果是，显示通知弹窗
3. 引导用户重新选择词库

**代码示例**（前端 config.js）：
```javascript
// 在 onShow 或 loadConfig 后添加
const OLD_LIBRARIES = ['托福 9000 词 Word', '词汇总编 9800 带音标 + 词汇 9400', ...];
const hasOldLibrary = this.data.config.vocab_library.some(lib => 
  OLD_LIBRARIES.includes(lib)
);

if (hasOldLibrary) {
  wx.showModal({
    title: '🎉 词库已更新',
    content: '词库列表已升级，请重新选择你想学习的词库',
    confirmText: '去选择',
    showCancel: false,
    success: () => {
      // 自动滚动到词库选择区域
      wx.createSelectorQuery().select('.library-selector')
        .boundingClientRect(rect => {
          wx.pageScrollTo({ scrollTop: rect.top });
        }).exec();
    }
  });
}
```
