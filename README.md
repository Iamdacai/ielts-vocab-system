# 雅思智能背单词系统

## 项目概述
- **目标**：基于剑桥雅思1-18词汇的智能背单词小程序
- **核心功能**：灵活配置学习计划 + 艾宾浩斯遗忘曲线复习
- **技术栈**：Node.js后端 + 微信小程序前端

## 功能需求
### 学习配置
- ✅ 每周新词学习频率（1-7天/周）
- ✅ 每次学习新词数量（可配置）
- ✅ 复习提醒时间（可配置）

### 智能复习
- ✅ 基于艾宾浩斯遗忘曲线的抗遗忘复习
- ✅ 自动安排每日复习课程
- ✅ 错题重点强化

### 词汇范围
- ✅ 剑桥雅思1-18完整词汇库
- ✅ 词汇分级（高频/中频/低频）
- ✅ 雅思真题例句
